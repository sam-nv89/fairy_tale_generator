# План развития (Roadmap) - Генератор Аудио-Сказок

## Фаза 1: MVP (Завершено) ✅
- [x] Базовая настройка интерфейса Streamlit.
- [x] Интеграция с Google Gemini API для генерации сказок.
- [x] Интеграция синтеза речи (переход с gTTS на качественный Edge-TTS).
- [x] Управление «Секретами» для публичного доступа (без ввода ключа пользователем).
- [x] Кастомный HTML5 Аудио Плеер с перемоткой и современным дизайном.
- [x] Выбор голоса (Мужской/Женский).

## Фаза 1.5: Лендинг и UI (Завершено) ✅
- [x] **Landing Page 2.0**: Полный редизайн с glassmorphism, mesh-градиенты и анимации.
- [x] **Prompt Engineering 2.0/2.5**: Адаптивная генерация по возрастным группам.
- [x] **Мультивалютность**: Автоопределение валюты (RUB, USD, EUR, KZT, BYN, UZS).
- [x] **Авторизация**: Интеграция Supabase Auth (вход/регистрация).
- [x] **Кастомный скроллбар**: Proximity-detection, авто-скрытие.
- [x] **Роутинг**: Гости → Лендинг, Авторизованные → Генератор.

## Фаза 2: Улучшение пользовательского опыта (В планах)
- [ ] **История сказок**: Сохранение сгенерированных историй в сессии, чтобы можно было вернуться назад.
- [ ] **Скачивание**: Кнопки для скачивания текста (.txt) и аудио (.mp3).
- [ ] **Фоновая музыка**: Наложение тихой колыбельной на голос (используя `ffmpeg`).
- [ ] **Мобильная версия**: Улучшение адаптивности плеера для маленьких экранов.

## Фаза 3: "Pro" функции
- [ ] **Клонирование голоса**: Возможность для родителей клонировать свой голос (через ElevenLabs API).
- [ ] **Иллюстрации**: Генерация обложки для каждой сказки с помощью Gemini/Midjourney.
- [x] **Аккаунты**: Система входа и интеграция Supabase (Базовая авторизация реализована).
